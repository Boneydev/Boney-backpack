# Boney-backpack

A simple backpack script for Qbox using ox_inventory.

## Features
- Buyable backpack item
- Usable item
- Opens personal stash
- Backpack visual when used

## Requirements
- qbx_core
- ox_inventory
- ox_lib

## Installation
1. Place `Boney-backpack` in your resources folder
2. Add to server.cfg:
3. ensure Boney-backpack

4. 3. Add item to items file
4. Add item to shop
